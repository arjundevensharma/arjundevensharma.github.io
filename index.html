Green Meter

Awarded "Best of Air Challenge" at KuriusHacks

Chrome extension that tracks and visualizes the equivalent carbon emission yield that was produced from a given browsing session. Provides real-time energy consumption and carbon footprint data. Contextualizes data by showing the number of trees needed to offset the carbon released from the user's browsing and a simplified visual representation. Integrates the ElectricityMaps API for real-time carbon intensity data based on geographic location.

Climate Criminals

Honourable mention at HackSwift

Browser extension that identifies and highlights unsustainable companies that are contributing to climate change as you browse. In doing so, it points out a specific, objective facet of their behaviour that is contributing to climate change and even supports this information with a link. The extension also suggests eco-friendly alternatives and supports reliability through accessing the Greenhouse 100 Polluters Index 2023 and recent news sources.

ReadWise

Awarded 2nd Place Overall at HackOrange

Reading literacy platform that provides a reading test that presents a passage for students to read through, followed by a comprehension quiz. After submitting the quiz, the user is provided with their reading speed and a book recommendation based on their performance. Recommendations encompass a wide range of reading levels and genres. Aims to make reading more accessible.

The Last Parking Spot

Musical Chairs-inspired Unity 2D racing game w/ AI functionality. Players compete to park their cars first when the music stops. Featuring realistic physics, collision detection and response mechanics, a machine learning algorithm based on the A* search algorithm for post-win pathfinding animations. Utilizes a robust win condition involving the randomization of the parking sign's location on the track when the music stops to reward strategic thinking. 214073 total lines of code.

Siri Emulation Model

93.7% accurate machine learning voice command intent classification model using TensorFlow and BERT. The model achieves its function by performing token-level classification to extract named entities (NER) from the given voice command using a BIO tagging scheme and using a trained BERT model to extract features of the voice command including dropout, intent classifier, and slot classifier layers to predict the overall intent of the voice command based on extracted named entities.
